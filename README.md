## рҹ’» Git Rules

### рҹ“Ң Commit

    1. м ңлӘ©мқ„ 50кёҖмһҗ лӮҙлЎң м ңн•ң
    2. м ңлӘ© лҒқм—җ л§Ҳм№Ён‘ң л„Јм§Җ м•ҠмқҢ
    3. м–ҙл–»кІҢ ліҙлӢӨлҠ” л¬ҙм—Үкіј мҷңлҘј м„ӨлӘ…

### рҹ“Ң Commit Structure

<p>н—ӨлҚ”лҠ” н•„мҲҳмқҙл©°, лІ”мң„(scope), ліёл¬ё(body)мқҖ м„ нғқмӮ¬н•ӯ</p>

```
<type>(<scope>): <subject>          -- н—ӨлҚ”
<BLANK LINE>                        -- л№Ҳм№ё
<body>                              -- ліёл¬ё(м„ нғқ)
```

### рҹ“Ң Commit Type

```
feat : мғҲлЎңмҡҙ кё°лҠҘм—җ лҢҖн•ң м»Өл°Ӣ (*)
fix : лІ„к·ё мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ (*)
style : мҪ”л“ң мҠӨнғҖмқј нҳ№мқҖ нҸ¬л§· л“ұм—җ кҙҖн•ң м»Өл°Ӣ (*)
refactor :  мҪ”л“ң лҰ¬нҢ©нҶ л§Ғм—җ лҢҖн•ң м»Өл°Ӣ (*)
build : л№Ңл“ң кҙҖл Ё нҢҢмқј мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ (*)
chore : к·ё мҷё мһҗмһҳн•ң мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ (*)
ci : CIкҙҖл Ё м„Өм • мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ
docs : л¬ём„ң мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ (*)
test : н…ҢмҠӨнҠё мҪ”л“ң мҲҳм •м—җ лҢҖн•ң м»Өл°Ӣ
```
