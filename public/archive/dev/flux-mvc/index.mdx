## 📌 Flux 란?

MVC 패턴의 복잡한 데이터 흐름 문제를 해결하는 개발 패턴(Unidirectional Data Flow)

![https://user-images.githubusercontent.com/38034518/228743214-f57100e8-631f-4730-9543-e719ff5993c0.png](https://user-images.githubusercontent.com/38034518/228743214-f57100e8-631f-4730-9543-e719ff5993c0.png)

1. `action`: 화면에서 발생하는 이벤트 또는 사용자의 입력
2. `dispatcher`: 데이터를 변경하는 벙법, 메서드
3. `model`: 화면에 표시할 데이터
4. `view`: 사용자에게 비춰지는 화면
