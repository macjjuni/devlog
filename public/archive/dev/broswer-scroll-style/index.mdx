주로 Mac과 iOS를 주로 사용하면서 데스크톱은 크롬, 모바일은 Safari 브라우저를 사용한다. 블로그에 들어올 때마다 윈도우의 투박한 스크롤바가 거슬렸고 이번 기회에 스크롤 디자인을 적용해 봤다.

브라우저마다 스크롤바 스타일링을 위한 CSS 코드가 다르니 주의해야 하고 OS와 브라우저별 스타일 통합을 원한다면 라이브러리를 사용하는 편이 좋으니 관련 라이브러리도 찾아 기록해 본다.

## 🎨 CSS

```css
/* Chrome, Edge, and Safari */
*::-webkit-scrollbar {
  width: 6px;
  transition: 0.3s ease;
}
/* Scrollbar Background Style */
*::-webkit-scrollbar-track {
  background: #eee;
}
/* Scrollbar Body Style */
*::-webkit-scrollbar-thumb {
  background-color: black;
}
/* Firefox */
* {
  scrollbar-width: thin;
  scrollbar-color: black #eee;
				/* body background */
}
```
